Introduction to GitHub

What is Version Control?

What is “version control”, and why should you care? Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later. For the examples in this workshop, you will use software source code as the files being version controlled, though in reality you can do this with nearly any type of file on a computer.

What is Git?

Git is a distributed version-control system for tracking changes in source code during software development.

What is GitHub?

TLDR: Cloud-based Git.

GitHub is a Git repository hosting service, but it adds many of its own features. While Git is a command line tool, GitHub provides a Web-based graphical interface. It also provides access control and several collaboration features, such as a wikis and basic task management tools for every project.
