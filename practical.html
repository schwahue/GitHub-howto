Step 1. Creating a Reposistory

1. Click on the + button on the top right corner and then select New reposistory.

2. Name your reposistory and write a short description.

3. Select Initialize this repository with a README.

Step 2. Create a branch

1. Click on master branch

2. Create a new branch

Step 3. Make and commits changes

1. Select the "README.md" file

2. Make some changes

3. Commit the changes

Step 4. Make a pull request

1. Go to "Pull requests" on the navigation bar

2. Click on the green "New pull request" button

3. Create new pull request

4. Merge the pull request
